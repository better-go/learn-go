## 示例服务：

- run:

```ruby

cd learn-go/

# go mod tidy:
task try:zero:tidy 

# run:
task try:zero:run:api

```


### api 示例：

- new：

```ruby

goctl api new hello

```

### rpc 示例：

- new：

```ruby

goctl rpc new rpc
```
